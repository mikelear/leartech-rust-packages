# leartech-rust-packages

Auto-generated client SDKs published from leartech service repos via the
[`openapi-generation`](https://github.com/mikelear/leartech-pipeline-catalog/tree/main/tasks/openapi-generation)
catalog task. Each subdirectory is one service's generated client.

Do not hand-edit — every release of a consumer service overwrites its
subdir via auto-merging PR.
